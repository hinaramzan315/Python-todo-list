# 📝 To-Do List App (Python)

A simple console-based To-Do List application built in Python as **Project 1** of my Industrial Training Internship at **DecodeLabs (Batch 2026)**.

## 📌 About

This project focuses on the fundamentals of data management using Python's built-in data structures — the same core logic that powers real-world databases at scale. Before building complex systems, it's essential to understand how to store, retrieve, and manage multiple items efficiently.

## ✨ Features

- ➕ **Add Task** — Add new tasks to your to-do list
- 📋 **View Tasks** — Display all tasks with their index number
- 🔁 **Menu-driven loop** — Keep adding/viewing tasks until you choose to exit
- 🚪 **Exit option** — Cleanly end the program

## 🛠️ Concepts Used

- Python **Lists** (dynamic data storage)
- `append()` method for adding data
- `enumerate()` for clean, index-based iteration
- **Functions** for modular, reusable code
- `if __name__ == "__main__":` — standard Python entry-point pattern

## 🚀 How to Run

1. Clone this repository
   ```bash
   git clone https://github.com/hinaramzan315/todo-list-python.git
   ```
2. Navigate to the project folder
   ```bash
   cd todo-list-python
   ```
3. Run the script
   ```bash
   python todo_list.py
   ```
